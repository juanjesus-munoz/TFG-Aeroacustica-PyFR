# TFG: Simulación aerodinámica de instrumentos musicales tipo flauta mediante CFD con PyFR

Este repositorio contiene los archivos fuente y scripts de análisis desarrollados para mi
Trabajo de Fin de Grado, centrado en el estudio del comportamiento aerodinámico y aeroacústico
acoplado en instrumentos de bisel bajo régimen de bajo número de Mach, mediante simulación
numérica directa (DNS) con PyFR.
