# TFG: Simulación aeroacústica de alta fidelidad mediante CFD (PyFR)

Este repositorio contiene los archivos fuente y scripts de análisis desarrollados para mi Trabajo de Fin de Grado, centrado en el estudio del comportamiento aerodinámico y aeroacústico en régimen de bajo número de Mach.

### 📂 Contenido del Proyecto
*   **Mallas (.msh):** `caso_1.msh`, `caso_1_redondeado.msh`, `caso_2.msh` generadas en **Pointwise**.
*   **Configuración (.ini):** `configuracion.ini`, `configuracion_filtro.ini` para la ejecución en **PyFR**.
*   **Scripts de Análisis (.m):** `bisel_forces.m`, `probes.m`, `representacion_sondas.m` para el procesamiento de datos en **MATLAB**.

### 📄 Portafolio de Resultados
Puedes consultar el informe detallado con las gráficas de validación, análisis espectral y conclusiones aquí: [**Descargar Portfolio_TFG.pdf**](Portfolio_TFG.pdf)

### 🛠️ Stack Tecnológico
*   **CFD:** PyFR, Pointwise, ParaView.
*   **Análisis de datos:** MATLAB, Python.
*   **Entorno:** VS Code.
